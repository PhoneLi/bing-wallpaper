# bing-wallpaper
拉取bing首页的背景壁纸，然后设置成mac的默认壁纸。

# 使用说明
1: 运行下面命令 
> $> nohup ./auto.sh &
  
  这时候，程序会每隔30分钟检查本地是否已经有今天的bing壁纸。
  如果没有，则拉取最新bing首页的壁纸，然后设置本地桌面壁纸
  （如果本地有今天bing的最新壁纸，则不再重复拉取，也不会设置壁纸）
  
2:运行下面命令
  
> $> cp ./dark.png ~/Pictures/bing

# bing.sh的使用说明
1: 运行
> $> bing.sh dark

  这个命令可以把当前的桌面壁纸设置成全黑色。
  
2: 运行
> $> bing.sh bing-load
  
  这个命令可以立刻把当前的桌面壁纸设置成今天的bing壁纸。
